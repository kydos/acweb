# Security

Zenoh is **Security by Design** — written entirely in Rust for memory safety, with
authentication, encryption, and access control built into the protocol.

- [TLS Configuration](./tls.md) — mutual TLS for transport encryption
- [Access Control Lists](./acl.md) — fine-grained permission rules
