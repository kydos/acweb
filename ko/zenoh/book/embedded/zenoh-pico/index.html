Zenoh-Pico (C)

Zenoh-Pico is the C implementation of Zenoh for microcontrollers. It requires as little as 2 KB of RAM and runs on virtually any MCU with a C compiler.

Features

  • Implements the full Zenoh protocol (pub/sub, queryables)
  • MISRA-C compliant — limits potential for memory errors
  • Static analysis in CI
  • Supports FreeRTOS, Zephyr, Arduino, and bare-metal
