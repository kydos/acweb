The Zenoh Router

For scenarios where peers cannot directly reach each other — different networks, NAT, cloud-to-edge — Zenoh provides a router (zenohd).

Router Topology

Routers are not limited to chains — they can be interconnected in any topology, including mesh networks where each router has multiple peers. Zenoh's link-state routing algorithm automatically computes the optimal path through the graph and reroutes around failures.

Each hop adds minimal latency. Whether the deployment is a linear chain, a star, or a full mesh, the application code is identical — topology is a configuration concern, not an API concern.
